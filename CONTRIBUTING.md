# Contributing to KrishiAi

Thank you for considering contributing to the KrishiAi project! We welcome contributions from the community to improve this project.

## Guidelines for Contributing

### Code Style
- Follow the existing style and patterns used in the project.
- Use meaningful variable names and write clear comments.

### Commit Messages
- Use the present tense: "Add feature" not "Added feature".
- Limit the first line to 72 characters or less.
- Reference relevant issues if applicable (e.g., "Fixes #123").

### Testing Requirements
- Please write tests for any new features or bug fixes.
- Run the test suite before submitting your contribution to ensure everything works as expected.

### Pull Request Process
1. Fork the repository and create your branch from `main`.
2. Ensure your code passes all tests and adheres to the coding standards.
3. Open a pull request and provide a clear description of your changes.
4. Be open to feedback and willing to make changes if requested.

Thank you for contributing!
